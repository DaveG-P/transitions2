Transitions

If I use the term "all", what can I delete from the following code?

transion: background .3s, ease color .02 linear;

If a designer wants to use transitiions with the same effects then they would replace both "background .3s" and color, with "all". This allows to cover all the possible properties that can be transitioned.

Define the following timeing functions.transitioning elements:

"CSS Transitions allow propoert changes in CSS values to occur smoothly over a specified duration"

  1. Ease: Transition that goes from slow, to fast, then slow.
  2. Linear: Transition with a consistent speed.
  3. Ease-In: A slow start transition.
  4. Ease-Out: Transition that both begins and ends slow
  5. Cubic-Bezzier: Transition with custome values defined in the Cubic Bezzier function.
